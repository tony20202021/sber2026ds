# Plan: Предсказание пола по транзакциям (Sber 2026 DS)

## Цель

Предсказать пол клиента (`0` — женщина, `1` — мужчина) по истории банковских транзакций.

**Целевые метрики:** ROC AUC > 0.88 и Accuracy > 0.80

**Датасет:** [`mks-logic/gender_prediction`](https://huggingface.co/datasets/mks-logic/gender_prediction)
— 3 751 083 транзакций, 8 400 клиентов (70% train / 10% val / 20% test).

---

## Окружение

```bash
conda activate sber   # python 3.10 (amikhalev_language_learning_bot env)
# зависимости: pandas, numpy, scikit-learn, lightgbm, spacy, ru_core_news_md
```

```bash
python download_dataset.py             # скачать данные в ./data/
python solution.py                     # baseline (376 фич, 5-fold LGB)
python feature_importance.py           # анализ важности фич + сравнение cutoff
python compute_gender_embeddings.py    # precompute MCC/TR gender scores → results/gender_embeddings/
python solution_super_200.py           # лучшее решение (685 кандидатов → top-200)
```

## Задачи

- [x] **1. Conda окружение** `sber`, конвертация ноутбука в `solution.py`
- [x] **2. Целевые метрики** (AUC > 0.88, Accuracy > 0.80) — достигнуты в `solution_2026_04_06`
- [x] **3. Feature importance** — график top-50, оптимальный cutoff (top-100)
  - ветка: `solution_2026_04_15_feature_importance`
  - результаты: `results/solution_2026_04_06/`
- [x] **4. Сравнение моделей** — LogReg и RandomForest, метрики + время
  - ветки: `solution_2026_04_15_linear`, `solution_2026_04_15_random_forest`
- [x] **5. `main.py`** — ветка `solution_2021_old`
  - адаптация оригинального скрипта (vectorized features, XGBoost CV)
  - фичи: per-hour/day shares (42 total), night, pos/neg stats
  - AUC=0.680, acc=0.644
- [x] **6. LightAutoML** — ветка `solution_2026_04_15_lama`
  - `lightautoml==0.4.2` (Sber AI Lab), blend LinearL2+LightGBM, timeout=300s
  - AUC=0.877, acc=0.782 — хуже ручного решения
- [x] **7. Best-from-all** — ветки `solution_2026_04_15_best_from_all_130/200`
  - _130: top-100 + 30 фич из solution_2021_old → AUC=0.879, acc=0.798
  - _200: top-170 + 30 фич из solution_2021_old → AUC=0.882 ✓, acc=0.801 ✓
- [x] **8. Holiday features** — ветки `solution_2026_04_15_holidays_200/230`
  - 10 праздников × (mean/std/near7_ratio) = 30 фич
  - вывод: праздники дают +0.001 AUC поверх базы, не вытесняя её
- [x] **9. MCC × time interactions** — ветка `solution_2026_04_15_mcc_interactions`
  - top-20 MCC × (morning/afternoon/evening/night/weekend_ratio + q25/q75/median) = 160 фич
  - 50/160 вошли в top-200; AUC=0.883 ✓, acc=0.802 ✓
- [x] **10. Gender text embeddings** — ветка `solution_2026_04_15_gender_embedding`
  - spaCy `ru_core_news_md` (300d Word2Vec): cosine similarity описаний MCC/TR
    с мужским/женским словесным центроидом
  - 9 фич: mcc/tr_gender_diff_mean/std, male/female_sim_mean, amount-weighted diff
  - все 9 вошли в top-200; AUC=0.881 ✓, acc=0.803 ✓
- [x] **11. Super-ensemble** — ветка `solution_2026_04_15_super_200`
  - 685 кандидатов (base+hour_shares+holidays+MCC×time+MCC_ext+gender_emb)
  - top-200 → **AUC=0.882 ✓, acc=0.807 ✓ — лучшая accuracy за все эксперименты**
  - top-100 → AUC=0.874, acc=0.789 — ниже порога (100 фич недостаточно)
  - состав top-200: base=95, hour_shares=24, MCC×time=35, MCC_depth=12, holidays=25, gender_emb=9

## Feature Engineering

### Типы признаков и их вклад

| Группа | Кол-во | Описание | В super top-200 |
|---|---|---|---|
| A. Base (solution.py) | 376 | Amount stats, temporal, per-MCC/TR, target encoding | 95 |
| B. Hour shares + pos/neg | 30 | Per-hour normalized shares (0-23), pos/neg min/max/median | 24 |
| C. MCC × время | 160 | top-20 MCC × morning/afternoon/evening/night/weekend/q25/q75/median | 35 |
| D. MCC depth | 80 | top-20 MCC × neg_ratio, max_amount, amount_cv, cnt_ratio | 12 |
| E. Праздники | 30 | 10 праздников × mean/std/near7_ratio до ближайшего вхождения | 25 |
| F. Gender embeddings | 9 | spaCy cosine sim описаний MCC/TR с male/female centroid | 9 |

### Топ-5 признаков по importance (из feature_importance.csv)
`mcc_code_te`, `dow0_ratio`, `dow6_ratio`, `hour_std`, `afternoon_ratio`

### Базовые группы (solution.py, 376 фич)

| Группа | Признаки |
|---|---|
| Глобальные amount | count, sum, mean, std, min, max, q25, q75, median, cv, range |
| Pos/neg split | count, sum, mean, std + pos_ratio, pos_neg_ratio |
| Diversity | nunique MCC, nunique tr_type, term_missing_ratio |
| Temporal | hour mean/std, dow_ratio×7, weekend/morning/afternoon/evening/night ratio |
| Active span | active_days, tx_per_day, amount_per_day, avg/std days between tx |
| Recency | recent_ratio, recent/early amount_mean, amount_trend |
| Entropy | MCC entropy, tr_type entropy |
| Weekend vs weekday | weekend/weekday amount mean, diff |
| Amount buckets | 6 бинов: very neg / neg / ~0 / small pos / mid pos / large pos |
| Per-MCC (60 кодов) | cnt, mean, std, share |
| Per-tr_type (25 типов) | cnt, mean, share |
| Target encoding (OOF) | mcc_code_te, tr_type_te — p(male) |
